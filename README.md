# soat23-gp14-produto

## Descrição

Este Microserviço é reponsável por gerenciar os produtos do catálogo na Plataforma de Pedidos. Ele possuis as funções de:
- Criar produto
- Atualizar produto
- Deletar produto
- Consultar produto

## Pré-requisitos
- Node
- MongoDB
- 
## Executando a aplicação
- Faça o download do repositório através do arquivo zip ou do terminal usando o git clone
- Acesse o diretório do projeto pelo seu terminal
- Instale as dependências:
```bash
$ yarn install
```
- Inicie a aplicação
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test
Execute os testes:
```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
