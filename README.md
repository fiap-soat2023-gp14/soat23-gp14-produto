# soat23-gp14-produto

## Description

Product microservice responsible for managing the product catalog.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
## Docker

Build the image and run the container:

```bash
docker build -t app-fiap .

docker run --name app-fiap  -p 8080:8080 -d app-fiap
```
